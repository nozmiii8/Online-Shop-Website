## ğŸš€ Cara Menjalankan Aplikasi (Setup & Run)

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di lingkungan lokal Anda.

### 1. Persiapan Database

Pastikan server **MySQL** Anda sudah berjalan (misalnya, melalui XAMPP atau MAMP).

* Buat database baru (misalnya bernama `db_tokoonline`).
* Impor file *schema* dan data awal (jika tersedia) ke dalam database tersebut.
* Perbarui kredensial koneksi database di file konfigurasi PHP Anda (biasanya di `config.php` atau sejenisnya).

### 2. Menjalankan Server Pengembangan

1.  Buka Terminal atau Command Prompt.
2.  Navigasikan ke direktori utama proyek Anda (`cd /path/ke/folder/proyek`).
3.  Jalankan server pengembangan PHP:

    ```bash
    php -S localhost:8080
    ```
